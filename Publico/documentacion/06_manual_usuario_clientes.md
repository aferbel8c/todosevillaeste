# Manual de Usuario - Clientes

## 1. Qué es TodoSevillaEste

TodoSevillaEste permite encontrar negocios de Sevilla Este, consultar información actualizada, contactar, dejar reseñas, recibir notificaciones y pedir soporte.

## 2. Acceso

El usuario puede entrar desde navegador o desde la app Android cliente. La pantalla principal muestra negocios, buscador y accesos a cuenta.

## 3. Buscar negocios

1. Entrar en la página principal.
2. Usar el buscador.
3. Escribir nombre, categoría o palabra relacionada.
4. Abrir la ficha del negocio deseado.

En cada negocio se puede consultar:

- Nombre.
- Categoría.
- Dirección.
- Teléfono.
- Web.
- Email público si está disponible.
- Descripción.
- Fotos.
- Horarios.
- Mapa.
- Reseñas.

## 4. Crear cuenta

1. Entrar en `register.html`.
2. Indicar usuario, email y contraseña.
3. Completar captcha/antibot si aparece.
4. Confirmar código de email si se solicita.
5. Acceder a la cuenta.

Recomendaciones:

- Usar una contraseña segura.
- Usar un email real para recuperación.
- No compartir la cuenta.

## 5. Iniciar sesión

1. Entrar en `login.html`.
2. Escribir usuario/email y contraseña.
3. Completar verificación si se solicita.
4. Marcar recordar sesión solo en dispositivos propios.

Si está disponible, también puede usarse Google o Microsoft.

## 6. Perfil

Desde el perfil el usuario puede:

- Ver datos de cuenta.
- Cambiar nombre visible.
- Cambiar email.
- Cambiar contraseña.
- Subir o eliminar avatar.
- Consultar reseñas propias.
- Solicitar eliminación de cuenta.

## 7. Reseñas

Para dejar una reseña:

1. Iniciar sesión.
2. Abrir la ficha de un negocio.
3. Ir a la sección de reseñas.
4. Elegir puntuación.
5. Escribir descripción.
6. Adjuntar fotos si procede.
7. Enviar.

El usuario puede editar o eliminar reseñas propias. También puede votar reseñas y reportar contenido que considere inapropiado.

Buenas prácticas:

- Describir experiencias reales.
- No publicar insultos, datos privados ni contenido falso.
- Adjuntar fotos solo si se tiene derecho a publicarlas.

## 8. Mensajes

El usuario puede abrir chats:

- Con un negocio desde su ficha.
- Con otro usuario desde el perfil público si la opción está disponible.

Acciones disponibles:

- Enviar texto.
- Adjuntar archivos permitidos.
- Ver conversaciones.
- Renombrar chat.
- Bloquear conversación.
- Borrar mensajes propios.
- Ocultar chat.

## 9. Soporte

Para pedir ayuda:

1. Iniciar sesión.
2. Entrar en soporte.
3. Elegir asunto.
4. Escribir explicación.
5. Adjuntar archivos si ayudan.
6. Enviar ticket.

El usuario podrá seguir la conversación y recibir respuestas de administración.

## 10. Alta de negocio

Un cliente autenticado puede solicitar o crear un negocio:

1. Entrar en `alta_negocio.html`.
2. Completar nombre, categoría, descripción y contacto.
3. Indicar horarios.
4. Marcar ubicación en mapa.
5. Aceptar consentimiento de publicación de datos de contacto si aplica.
6. Subir foto principal.
7. Enviar.

Según configuración, administración podrá revisar o activar el negocio.

## 11. Notificaciones

El usuario puede recibir:

- Notificaciones web si acepta permisos del navegador.
- Notificaciones Android si usa la app cliente y está configurado FCM.

Se recomienda aceptar notificaciones solo en dispositivos propios.

## 12. Descargar APK

La página de versiones permite descargar la última app cliente:

- `versiones_cliente.html`: cliente.
- `versiones.html`: cliente y admin si se muestran.

## 13. Privacidad

El usuario puede consultar:

- Protección de datos.
- Aviso legal.
- Política de cookies.
- Términos y condiciones.

Puede solicitar información, rectificación o eliminación de datos mediante los canales indicados por la plataforma.
