# Manual de Negocios - TodoSevillaEste

## 1. Objetivo

Este manual explica cómo un negocio usa TodoSevillaEste para aparecer en el directorio, mantener su ficha actualizada y comunicarse con clientes.

## 2. Alta de negocio

El alta se realiza desde `alta_negocio.html`.

Datos recomendados:

- Nombre comercial correcto.
- Categoría precisa.
- Dirección completa.
- Teléfono de atención.
- Web oficial.
- Email de contacto.
- Descripción clara.
- Frase inicial breve.
- Horarios actualizados.
- Ubicación en mapa.
- Foto principal de calidad.

## 3. Consentimiento de contacto

Si se publican teléfono, email o web, el negocio debe tener autorización para publicar esos datos. El formulario solicita consentimiento cuando procede.

## 4. Acceso al panel de negocio

El propietario entra en `negocios_gestion.html` con su cuenta.

Requisitos:

- Cuenta autenticada.
- Rol `business`.
- Negocio vinculado.

Si el acceso falla, revisar:

- Usuario incorrecto.
- Falta de rol negocio.
- Negocio no vinculado.
- Sesión caducada.

## 5. Editar ficha

Desde el panel se puede modificar:

- Nombre.
- Categoría.
- Dirección.
- Teléfono.
- Web.
- Email público/contacto.
- Descripción.
- Frase inicial.
- Horarios.
- Días especiales.
- Coordenadas.
- Fotos.

Los cambios afectan a la ficha pública cuando el negocio está activo.

## 6. Horarios

Los horarios se organizan por días y tramos.

Ejemplo:

- Lunes: 09:00-14:00 y 17:00-20:30.
- Domingo: cerrado.

Buenas prácticas:

- Mantener festivos como días especiales.
- Revisar horarios de vacaciones.
- Evitar textos ambiguos si el formulario permite tramos estructurados.

## 7. Días especiales

Los días especiales sirven para cierres excepcionales, fiestas, vacaciones o aperturas puntuales. Deben revisarse periódicamente para no mostrar información caducada.

## 8. Fotos

Tipos:

- Foto principal: imagen destacada.
- Fotos extra: galería.

Recomendaciones:

- Usar fotos propias.
- Evitar imágenes con datos privados de clientes.
- No subir imágenes con derechos de terceros sin autorización.
- Mantener buena resolución y peso razonable.

## 9. Mensajes

Los clientes pueden iniciar chat con el negocio desde la ficha. El negocio debe responder de forma profesional, evitando pedir datos sensibles innecesarios.

## 10. Reseñas

Los usuarios pueden publicar reseñas. El negocio debe:

- Revisarlas periódicamente.
- Responder por canales adecuados si existe la funcionalidad.
- Solicitar moderación solo cuando haya insultos, contenido falso, datos personales o abuso.
- No intentar manipular valoraciones.

## 11. Activación y desactivación

Administración puede activar o desactivar negocios. Un negocio inactivo puede no aparecer públicamente o mostrar restricciones.

Motivos típicos:

- Datos incompletos.
- Duplicado.
- Solicitud del propietario.
- Incumplimiento de condiciones.
- Revisión administrativa.

## 12. Recuperación de acceso

Si el propietario pierde acceso:

1. Usar recuperación de contraseña.
2. Revisar email asociado.
3. Contactar con soporte.
4. Aportar datos que acrediten relación con el negocio.

## 13. Buenas prácticas para negocios

- Mantener contacto actualizado.
- Revisar horarios al menos una vez al mes.
- Usar descripción clara y sin exceso de mayúsculas.
- Publicar fotos reales.
- Responder mensajes con rapidez.
- Solicitar soporte si hay datos incorrectos.
