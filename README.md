TodoSevillaEste es un ecosistema digital integral diseñado para conectar a los vecinos, comercios y administradores 
del barrio de Sevilla Este. La plataforma facilita el descubrimiento de negocios locales, la interacción comunitaria y la gestión profesional del tejido comercial.

Actualmente, se han subido las últimas versiones de todos los componentes del sistema, incluyendo la web pública, el panel de administración y las aplicaciones móviles.

Funcionalidades Clave:
Directorio Inteligente: Búsqueda avanzada de negocios con fichas detalladas, horarios y geolocalización.
Interacción Vecinal: Sistema de reseñas, votaciones y mensajería privada entre usuarios y comercios.
Gestión para Negocios: Panel exclusivo para propietarios donde pueden actualizar su información, galerías y horarios.
Asistente con IA: Chatbot inteligente integrado que responde consultas basadas en información pública real del barrio.
Administración Total: Herramientas avanzadas para la gestión de usuarios, moderación de contenidos y control técnico del servidor.

Stack Tecnológico:
Backend: Python con FastAPI, encargado de la lógica de negocio, seguridad y API REST.
Frontend: Arquitectura mixta que combina una capa productiva en HTML/JS con una migración progresiva a una SPA moderna en Vue 3 y Vite.
Móvil: Aplicaciones para Android basadas en tecnología WebView y soporte completo para PWA.
Base de Datos: MariaDB / MySQL para la persistencia relacional.
